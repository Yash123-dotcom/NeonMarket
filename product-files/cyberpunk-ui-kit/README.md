# Cyberpunk UI Kit

## What's Included

### Components (100+ Elements)

- **Buttons**: 25 different cyberpunk-styled buttons
- **Forms**: Login, signup, contact forms with neon styling
- **Navigation**: Futuristic nav bars and menus
- **Cards**: Data cards, profile cards, info panels
- **Modals**: Popup windows and dialogs
- **Icons**: 50+ cyberpunk-themed icons

### File Formats

- **Figma File**: `cyberpunk-ui-kit.fig`
- **Sketch File**: `cyberpunk-ui-kit.sketch`
- **Adobe XD**: `cyberpunk-ui-kit.xd`
- **CSS/SCSS**: Ready-to-use stylesheets
- **React Components**: JSX components for React projects

### Color Palette

- Primary: #00FFFF (Cyan)
- Secondary: #FF00FF (Magenta)
- Accent: #FFFF00 (Yellow)
- Background: #0A0A0A (Dark)
- Text: #FFFFFF (White)

### Typography

- Primary Font: Orbitron
- Secondary Font: Rajdhani
- Code Font: Fira Code

## Usage Instructions

1. Import the Figma/Sketch file into your design tool
2. Copy components to your project
3. Customize colors and text as needed
4. Export assets for development
